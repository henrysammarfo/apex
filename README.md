# APEX — Autonomous Portfolio Execution Agent

AI-driven RWA portfolio management on HashKey Chain. APEX deploys autonomous agents that monitor Chainlink price feeds, reason about risk, rebalance on-chain positions, and settle payments through HSP — with no human in the loop after setup.

## Features

- **4-Agent Architecture** — Monitor, Decision, Execution, and Settlement agents working in concert
- **Vault Configuration** — Set risk parameters, allocation targets, and rebalancing thresholds
- **Deposit & Withdraw** — Seamless on-chain fund management with real-time tracking
- **HashKey Wallet Connect** — Native integration with HashKey Chain (ID 133)
- **Telegram Alerts** — Configurable notifications for agent actions and portfolio changes
- **Onboarding Tour** — Interactive walkthrough for first-time users

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **UI**: shadcn/ui, Framer Motion, Recharts
- **Routing**: React Router v6
- **State**: React Query, React Context

## Getting Started

```bash
npm install
npm run dev
```
